🔑 Générateur de mot de passe (hash)

Cet outil fonctionne entièrement dans ton navigateur, hors ligne — ton mot de passe n'est envoyé nulle part. Tape-le, récupère le hash généré, et colle-le dans admin.html.

Clique sur le texte pour le sélectionner, puis copie-le (Ctrl+C).

Ce hash est une empreinte à sens unique : impossible de retrouver le mot de passe à partir de lui. C'est ce hash (et non le mot de passe) qui sera visible dans le code de admin.html.
