Aperçu en direct

Créer le jeton GitHub : GitHub → ton avatar → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Choisis « Only select repositories » et sélectionne uniquement ce repo, puis dans Permissions → Contents, choisis « Read and write ». Copie le jeton généré (il ne sera plus jamais réaffiché) et colle-le dans les réglages ci-dessus.
